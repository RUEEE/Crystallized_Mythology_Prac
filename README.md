- 直接放入游戏文件夹下后打开 Prac.exe
- 适用于 1.03a (http://www1.iza-yoi.net/)

## 功能:
- 按 **backspace** 后
+ **F1** 可以开无敌
+ **F2** 禁用X键
+ **F3** 永续bgm
- 进入练习界面后可以选择从各种地方开始(甚至居然有ex，，，)
- 修复了日文乱码的问题

## 如何播放使用练习器的rep:
- 进入练习模式, 将参数调到rep的对应参数, 然后退出, 进入rep
- 注意按 backspace 后显示 pracmode: true, 则可播放
